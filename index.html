Migração do modelo legado de valoração para uma nova lógica baseada na tabela de mercado da Jovedata, conversão de pixels para cm² e o fator de audiência do Similarweb. Este painel compara os dois modelos sobre a massa de dados de exemplo e mede o impacto real nos valores cobrados dos clientes.

As duas lógicas

Modelo antigo × modelo novo

O modelo legado divide o preço interno por uma constante fixa. O novo deriva o preço por cm² do próprio banner e o escala pela área capturada e pela audiência.

Legado · a confirmar

Modelo Antigo

Valor = ( Preço_Interno / 17,55 ) × Área_Capturada_cm²
Preço vem do sistema interno (não do mercado).
Constante fixa 17,55 — não reflete tamanho real do banner nem audiência.
Novo · Jovedata + Similarweb

Modelo Novo

Valor = ( Preço_Jovedata / ( (Larg_px×Alt_px) / 1428,84 ) ) × Área_Capturada_cm² × Fator_Similarweb
Preço de mercado por cm² do banner real.
Audiência entra explicitamente via Similarweb.
Sensível: banner minúsculo ⇒ denominador → 0 ⇒ valor dispara.

Gargalos & regras de transição

Onde o modelo quebra — e como travar

Quatro padrões de risco identificados na massa de dados, com a trava proposta para cada um.

÷ 0 Dimensão zerada

977 anúncios (25,6%) vêm com altura ou largura em 0 px — vídeos, CPM, formatos sem medida. O denominador zera e a fórmula nova explode/quebra.

Trava: se Larg×Alt = 0 → não aplica modelo novo; mantém o valor legado como fallback até a dimensão ser preenchida.

↑↑ Banner minúsculo dispara

Banners 1×1, 20×120 etc. geram preço por cm² altíssimo. Ex.: "Publi Editorial" 1×1 px a R$17.538 sobe 25.076× sobre o antigo.

Trava (teto): Valor Novo limitado a 2× o valor antigo. Acima disso, sinaliza para revisão manual.

audiência = 0 Portal sem Similarweb

Portais novos ou de nicho podem vir sem dado de audiência. Fator 0 zera o valor do anúncio indevidamente.

Trava: Fator ausente/zero → assume fator neutro = 1,0 e marca o portal para auditoria de audiência.

↓↓ Queda generalizada

Mediana de 0,42× indica que muitos clientes seriam subprecificados, com perda de receita relevante na virada.

Trava (piso): Valor Novo não cai abaixo de 0,5× o valor antigo, suavizando a transição comercial.

Recomendação para bater o martelo

O modelo é viável — com governança

A nova lógica é tecnicamente sólida e traz o mercado (Jovedata) e a audiência (Similarweb) para dentro do preço. Sozinha, porém, ela oscila demais. A viabilidade depende das travas:

  • Implementar as 4 travas (÷0, teto 2×, piso 0,5×, fator neutro) antes de qualquer cobrança real.
  • Tratar os 977 formatos sem dimensão junto à Jovedata — é o maior bloqueio (25,6% da base).
  • Migração faseada: rodar os dois modelos em paralelo (shadow) por 1–2 ciclos e cobrar pela banda travada.
  • Confirmar a constante 17,55 do modelo antigo e a densidade 1428,84 com a engenharia.
  • Negociar com a Jovedata SLA de preenchimento de dimensão e cobertura de portais de nicho.
